# 《星航棋域》Demo 试玩说明

## 如何打开

直接双击同目录下的 `index.html` 即可打开试玩。

如果浏览器限制本地文件，也可以在当前目录运行：

```powershell
python -m http.server 5173
```

然后打开：

```text
http://localhost:5173/
```

## 当前版本内容

这是第一版可玩原型，已经包含：

- 六边形星域地图
- 四方阵营
- 蓝方玩家操作，红/绿/黄由简单 AI 控制
- 截击机、轰炸艇、预警舰三类单位
- 行动力移动
- 战争迷雾
- 资源点占领
- 折跃门占领与跳跃
- 中央星门占领得分
- 简化战斗结算
- 20 回合赛季结算

## 基础目标

你操控蓝方，在 20 回合内尽量获得最高总收益。

主要得分方式：

- 占领中央星门：每回合 +10 分
- 占领折跃门：每回合 +1 分
- 占领资源点：每回合 +2 资源

最终结算会综合积分和资源。

## 操作方式

1. 点击蓝方舰队，查看可移动范围。
2. 点击蓝色光圈内的格子进行移动。
3. 站在资源点、折跃门或中央星门上时，点击右侧「占领」。
4. 站在己方折跃门上，且己方拥有另一座折跃门时，点击「折跃」。
5. 点击敌方单位可发动攻击。
6. 行动力用完后，点击「结束回合」。

## 单位说明

| 单位 | 定位 |
| --- | --- |
| 截击机 | 机动战斗单位，克制轰炸艇 |
| 轰炸艇 | 高攻击单位，适合争夺据点 |
| 预警舰 | 视野和支援单位，周围友军战斗更强 |

## 第一轮试玩重点

请重点观察：

1. 是否能在 3 分钟内理解目标。
2. 是否愿意主动争夺中央星门。
3. 折跃门是否真的有用。
4. 战斗结果是否看得懂。
5. AI 是否能制造一点压力。
6. 是否想再玩一局。

## 已知限制

- 当前不是完整 SLG，只是玩法原型。
- 暂时没有主城建设和科技树。
- 暂时没有真正联机。
- AI 较简单，只用于制造基础对抗。
- 美术是原型级表现，后续可替换为正式资产。
